# Git Homework #1 – Git Workflow and History Management

This repository contains my submission for the Git homework assignment.

The goal of this project was to practice Git workflow and history management by creating commits, recovering a lost commit, working with branches, and tagging the final version of the project.

## Tasks Completed
- Initialized a Git repository
- Added a project file and created several commits
- Cleaned up commit history
- Simulated and recovered a lost commit
- Created a branch from the recovered commit
- Tagged the final cleaned version as v1.0
- Pushed the repository to GitHub
